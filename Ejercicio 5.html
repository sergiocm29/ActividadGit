¡Bienvenido al Manual de Instalación de Visual Studio Code!

  1. ¿Cómo descargar Visual Studio Code?

    2. Accede al sitio oficial de Visual Studio Code:

    https://code.visualstudio.com/

    Selecciona la versión adecuada para tu sistema operativo (Windows, macOS o Linux).

    Haz clic en el botón "Download" para iniciar la descarga.

  2. ¿Cómo instalar Visual Studio Code?

    3. Abre el archivo descargado e inicia el asistente de instalación.

    Sigue las instrucciones del asistente para completar la instalación.

  3. Extensiones más importantes de Visual Studio Code

    4. Abre Visual Studio Code y haz clic en el icono de extensiones en la barra lateral (o usa el atajo Ctrl+Shift+X).

    Busca e instala las siguientes extensiones recomendadas:

    1. Live Server: Para previsualizar tus archivos HTML en tiempo real.

    2. Prettier: Para formatear tu código automáticamente.

    3. HTML CSS Support: Para mejorar el soporte de HTML y CSS.
